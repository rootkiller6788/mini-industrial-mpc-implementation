# Course Alignment — MPC for Integrating Process Level Control

| School | Course | Module Coverage |
|--------|--------|----------------|
| **MIT** | 6.302 Feedback Systems | DMC stability, terminal constraints |

| **Stanford** | ENGR205 Process Control | Industrial MPC, surge tanks |
| **Berkeley** | ME233 Advanced Control | State-space MPC, disturbance estimation |
| **Berkeley** | EE C128 Mechatronics | Embedded QP solvers (Hildreth) |
| **CMU** | 24-677 Adv Ctrl Systems | QP formulations, goldfarb-idnani |
| **Georgia Tech** | ECE 6550 Nonlinear Control | Integrating process dynamics |
| **Purdue** | ME 575 Industrial Control | Level control, surge tank sizing |
| **RWTH Aachen** | Industrial Control Systems | PLC implementation patterns |
| **Tsinghua** | Process Control Engineering | DMC, industrial MPC deployment |
| **ISA/IEC** | ISA-88, API 2350, IEC 61511 | Safety standards for level control |
